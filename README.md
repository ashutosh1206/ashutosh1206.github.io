# Ash's blog

## Creating a new post

To create a new blog post, run: `hugo new content/blog/<blog-post-slug>.md`

## Blog post template

```md
---
title: Testing testing
author: Ashutosh Ahelleya
date: 2024-07-10
categories:
  - Crypto
tags:
  - Test
slug: test
math: true
---

Testing something
```

## Updating the theme

`git submodule update --remote --merge`
